# Global Life Expectancy at Birth (2025) – Interactive Tableau Dashboard

## Overview
This project is an interactive data visualization created as part of the **Makeover Monday** trend. It explores global inequality in life expectancy at birth for the year 2025 through an engaging and user-friendly Tableau dashboard. The goal is to present complex demographic data in a clear, visual, and comparative format that supports data storytelling and analytical insights.

## Purpose
The dashboard was designed to:
- Highlight differences in life expectancy across countries.
- Provide both macro (global map) and micro (country ranking) perspectives.
- Enable users to quickly compare nations against the global average.
- Demonstrate practical skills in data visualization, dashboard design, and analytics.

## Features
- **Global Inequality Map:**  
  A choropleth world map that shows life expectancy distribution across countries.
  
- **Ranking & Extremes Bar Chart:**  
  Displays the highest and lowest performing countries, allowing easy comparison.
  
- **Global Average Reference Line:**  
  Helps quantify how far each country deviates from the world norm.
  
- **Interactive Filters & Tooltips:**  
  Users can hover and interact to view country-specific details such as rank and average years.

## Tools & Technologies
- **Tableau Public** – Dashboard creation and visualization  
- **Data Visualization Principles** – Colour theory, hierarchy, and storytelling  
- **Makeover Monday Dataset** – Source of life expectancy data
